1. O que é o Node.js?   R: é um ambiente de execução (runtime) JavaScript de código aberto e multiplataforma que permite executar códigos JavaScript no lado do servidor (backend), fora do navegador

2. O Node.js usa qual linguagem de programação?   R: JavaScript

3. O Node.js roda no navegador ou fora dele?   R: O Node.js roda fora do navegador (server-side). Ele é um ambiente de tempo de execução (runtime) baseado no motor V8 do Google Chrome, permitindo executar código JavaScript diretamente no sistema operacional para criar servidores, APIs e scripts de automação, sem depender de um navegador web para interpretar o código. 

4. Para que serve o Node.js?   R: principalmente para executar código JavaScript no lado do servidor (back-end), permitindo o desenvolvimento de aplicações web rápidas, escaláveis e em tempo real.

5. Qual comando é usado para executar um arquivo Node.js no terminal?   R:  Basta abrir o terminal, navegar até a pasta do arquivo e digitar

6. O que é o npm?   R:  é o gerenciador de pacotes padrão para o ambiente de execução Node.js

7. O que significa a palavra módulo no Node.js?   R: um módulo é um arquivo ou conjunto de arquivos JavaScript encapsulados que contêm funções, objetos ou variáveis reutilizáveis.

8. O Node.js é mais usado no front-end ou no back-end?   R: O Node.js é primordialmente utilizado no back-end

9. O que é o arquivo package.json?   R:  é o manifesto central e arquivo de configuração de projetos Node.js e JavaScript.

10. O Node.js é gratuito ou pago?   R:  é totalmente gratuito
